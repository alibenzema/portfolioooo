GSB - Gestion du Parc Automobile

Description : Application web permettant la gestion complète du parc automobile de GSB, remplaçant l'ancien système basé sur Excel. Développée pour les visiteurs médicaux et les gestionnaires.

Fonctionnalités principales :

  • Authentification et gestion des rôles (gestionnaire/administrateur, visiteurs médicaux)
  • Gestion complète des véhicules (CRUD, statuts)
  • Suivi détaillé des véhicules (entretiens, kilométrage, prêts)
  • Gestion des entretiens avec upload de documents
  • Gestion des visiteurs médicaux et leurs affectations
  • Système d'affectation des véhicules aux visiteurs

Technologies utilisées :

  • Framework Symfony 6
  • Doctrine ORM
  • MySQL
  • Bootstrap 5
  • JavaScript pour les interactions dynamiques
  • Système de fichiers pour les documents
